UX Case Study

Angeles de Medellín

UX Research Information Architecture UI Design Prototyping Responsive Web

Problem Statement

The Angeles de Medellín Foundation’s website was built on a basic template with poor navigation, unclear information hierarchy, and no mobile optimization. These issues made it difficult for users to donate, volunteer, or learn about the organization, directly limiting engagement and support.

Objectives & Goals

  • Redesign the website to improve usability, accessibility, and engagement.
  • Simplify navigation and content structure to help users quickly find key actions (donate, volunteer, learn).
  • Create a modern, mobile-friendly experience that communicates the foundation’s mission more effectively.
Image

Our Process

Discover

Conducted a heuristic evaluation and task-based walkthroughs to identify usability pain points.

Define

Analyzed insights from similar nonprofit sites to benchmark best practices and prioritize requirements.

Ideate

Sketched wireframes and created multiple visual iterations exploring different layouts and CTAs.

Design

Built a cohesive design system in Figma and prototyped responsive pages for testing.

User Needs

  • Clear, intuitive navigation to find donation and volunteer options easily.
  • Consistent visual hierarchy to make information scannable.
  • Mobile accessibility for users engaging from any device.
  • Emotional storytelling that reflects the foundation’s impact.

Sketches

Outcome

The redesign was built using Next.js and Tailwind CSS for performance and scalability. Two rounds of user testing confirmed improved clarity and navigation. The new design highlights storytelling and accessibility, aligning with the foundation’s mission. Stakeholders reported that the new site better communicates their goals and encourages donations and volunteer sign-ups.
